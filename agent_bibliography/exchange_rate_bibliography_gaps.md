# Exchange Rate Policy — Bibliography Gaps & Candidate Sources

**Purpose:** companion to [[exchange_rate_conceptual_map]]. That map's Coverage notes section originally identified seven areas where the 18-source base (`agent_bibliography/exchange_rate_policy/`) was thin or empty; as of 2026-07 the base has grown to 28 sources and Sections 1, 2, 5, and 7 are fully closed, Section 3 partially closed (see each section for status). This file lists concrete candidate sources to close the remaining gaps — not yet acquired, not yet processed into the map.

**How to use this:** pick a gap, source the PDF(s) (purchase/download/request), drop them into `agent_bibliography/exchange_rate_policy/` using the suggested filename (matches the existing `topic_description (Author, Year).pdf` convention so they slot into the Sources table cleanly), then process them into the conceptual map the same way the first 18 were done — one at a time.

Each entry notes which `#theme_cluster` it would land under, so the eventual conceptual-map integration is a light lift.

## 1. `#currency_crisis_dynamics` — thin theoretical spine (rests on 2 sources despite 4 case studies) — RESOLVED 2026-07

**Status: closed.** All three generations have now been acquired and processed into [[exchange_rate_conceptual_map]] as sources 19-23: Krugman (1979, first-gen original), Flood & Garber (1983/84, linear companion), Obstfeld (1984, second-gen self-fulfilling model — the working-paper precursor to the 1986 AER piece originally requested below), and both competing third-gen accounts, Krugman's balance-sheet/transfer-problem model (2000) and Chang & Velasco's open-economy bank-run model (1998). The two "Low/Nice-to-have" candidates below (Obstfeld 1994 synthesis, Garcia & Valpassos Brazil case study) were not acquired and remain open if further depth is wanted.

The base had crisis *mechanics* via textbook treatment (Krugman Ch. 18) and CFA's overview, but none of the original academic models. Original recommendation was to add across all three "generations":

## Suggested acquisition order

Sections 1, 2, 5, and 7 (currency crisis models; capital controls; Fama's UIP-failure econometrics; political economy) are now fully closed, and Section 3 (OCA theory) is partially closed — see above. Of the remaining open items, the next highest-priority picks are:

1. **Chamon & Garcia (2016)** (Section 2, Brazil angle) — a natural, low-effort next step given the base's Brazil mandate and the fact that the general capital-controls framework is now in place; this would be the direct empirical application to Brazil's own 2008-2012 IOF episode.
2. **Garman & Kohlhagen (1983)** (Section 4) — closes the one theory area (FX options/volatility) with no representation at all in the base.

Everything else can follow based on what LIS's near-term research agenda actually needs (e.g., the remaining OCA companions only matter if EMU/monetary-union analysis is on the roadmap; RMB depth only matters if non-Brazil EM work is planned).
